## Repository purpose

A public blog repository authored by a research engineer. Posts cover software engineering, AI / machine learning, AI4Science, and materials science. The repository is intended to be open-source and public-facing — assume every committed file will be read by external readers.

## Writing conventions

- **Language:** all prose, code, code comments, commit messages, and filenames are written in English. This applies even when discussing Japanese-language source material.
- **Voice:** concise but professional. The author prefers terse, technically precise prose over conversational filler. Avoid hedging, marketing tone, and rhetorical buildup — state the claim, then justify it.
- **Audience:** assume readers are technical (engineers and researchers). Do not over-explain standard CS / ML terminology; do explain domain-specific terms when crossing into AI4Science or materials science.
- **Code samples:** keep them minimal and runnable.
